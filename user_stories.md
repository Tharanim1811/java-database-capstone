# User Stories - Smart Clinic Management System

## Admin User Stories

1. As an admin, I want to create doctor accounts so that doctors can access the system.
2. As an admin, I want to create patient accounts so that patients can book appointments.
3. As an admin, I want to update user information so that records remain accurate.
4. As an admin, I want to deactivate user accounts so that unauthorized users cannot access the system.
5. As an admin, I want to view all appointments so that I can monitor clinic operations.
6. As an admin, I want to manage doctor schedules so that appointment availability is maintained.

## Patient User Stories

1. As a patient, I want to register for an account so that I can access clinic services online.
2. As a patient, I want to log in securely so that my personal information is protected.
3. As a patient, I want to view available doctors so that I can choose the right healthcare provider.
4. As a patient, I want to book an appointment so that I can receive medical care.
5. As a patient, I want to reschedule an appointment so that I can adjust to changes in my schedule.
6. As a patient, I want to cancel an appointment so that I can free the slot when I cannot attend.
7. As a patient, I want to view my appointment history so that I can track my past visits.

## Doctor User Stories

1. As a doctor, I want to log in to my dashboard so that I can manage my appointments.
2. As a doctor, I want to set my availability so that patients can book appointments during appropriate times.
3. As a doctor, I want to view my scheduled appointments so that I can prepare for patient visits.
4. As a doctor, I want to update appointment statuses so that records remain accurate.
5. As a doctor, I want to view patient information so that I can provide appropriate treatment.
6. As a doctor, I want to create and manage prescriptions so that patient treatment records are maintained.
7. As a doctor, I want to review patient history so that I can make informed medical decisions.
